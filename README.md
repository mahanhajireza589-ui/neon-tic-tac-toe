# 🎮 Neon Tic Tac Toe

A modern and visually appealing Tic Tac Toe game built with pure HTML, CSS, and JavaScript.

✨ Features a neon-inspired interface, smooth animations, and an interactive gameplay experience.

## 🚀 Live Demo

👉 https://mahanhajireza589-ui.github.io/neon-tic-tac-toe/

---

## ✨ Features

- 🎮 Classic Tic Tac Toe gameplay
- 🌈 Neon UI Design
- ⚡ Smooth animations
- 🏆 Winner detection
- 🎵 Background music
- 📱 Responsive design
- 🔄 Restart game anytime

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
